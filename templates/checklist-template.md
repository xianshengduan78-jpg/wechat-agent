# 发布前检查清单

## 内容检查
- [ ] 标题 ≤ 30 字，无冒号破折号
- [ ] 摘要 ≤ 120 字，无时间词
- [ ] 正文无禁词（颠覆/革命/赋能/值得关注的是...）
- [ ] 正文无时间词（今天/昨天/今日）
- [ ] 所有速览 ≤ 20 字

## 配图检查
- [ ] 封面图已生成（wide + square）
- [ ] 事件-截图一一对齐
- [ ] 无自制信息卡
- [ ] 截图格式符合要求

## 微信格式检查
- [ ] HTML 格式正确
- [ ] 图片路径已替换为微信 URL
- [ ] 素材已上传
- [ ] 草稿内容预览正常

## 最终确认
- [ ] 事实核对无误
- [ ] 立场适中
- [ ] 格式无异常
